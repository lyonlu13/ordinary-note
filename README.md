## 關於

### 它可以做什麼

&emsp;&emsp;在最初規劃時，我希望平凡筆記可以用來做任何事，包含計畫、紀錄、筆記等等。你可以把它當作一個工作區，新增任何需要的內容，或是單純透過複製貼上來加入任何你需要安排或釐清的資訊。

### 設計方向

&emsp;&emsp;為提供直覺的操作方式，所有資訊皆由方塊表示，並盡可能支援所有可能透過剪貼簿傳遞的內容。排版方式為自由放置，允許使用者設計自己的空間配置，不受限於文件格式。
